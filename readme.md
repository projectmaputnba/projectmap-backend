# Project Map Backend

## Installation

```bash
git clone git@github.com:projectmaputnba/projectmap-backend.git
cd projectmap-backend
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Tests

```bash
# unit tests
npm run test
```

## Lint

```bash
# eslint
npm run lint
```
